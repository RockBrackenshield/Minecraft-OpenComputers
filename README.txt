# Minecraft-OpenComputers
For lua programs from the minecraft mod OpenComputers

To use, click on the file, find and click the "Raw" button, and then use the command
wget 'full URL path' filename
This will download the text shown at 'full URL path' and place it in filename
files will be placed in the current working directory, unless you specify it to download else where by using:
wget 'full URL path' /dir/dir/dir/filename

The Computer requires an Internet card installed in order to download the file

Feel free to use these programs at will, just don't take credit for creating them, please.
